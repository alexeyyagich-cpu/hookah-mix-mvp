## Анализ конкурентов и цен рынка

### POS-системы в Германии (то, за что бары уже платят)

| Система | Цена/мес | TSE | Hookah-специфика | Примечание |
|---|---|---|---|---|
| **ready2order** | €7.90–€22.90 | Да (встроен) | Нет | 16,000+ клиентов DACH. Gastro-пакет €12.90. Наша интеграция уже готова |
| **orderbird** | €49 + TSE €139/год | Да | Нет | iPad-based. Куплен Nexi за $140-150M |
| **Lightspeed Restaurant** | от €69 (реально €300-800 с доп.) | Да | Нет | Дорогой, для крупных ресторанов |
| **SumUp POS** | от €99 | Да | Нет | Простой, не подходит для сложных операций |
| **Gastrofix** | по запросу | Да | Нет | Теперь Lightspeed |

**Вывод:** Бары платят €22-99/мес за POS, но ни одна система не умеет трекать табак по граммам, управлять рецептами миксов, считать себестоимость сессий.

|---|---|---|---|---|---|---|
| **Kavapp** | по запросу (trial бесплатно) | СНГ, Украина | Да | Да | Нет | Нет |
| **SkyService POS** | $17–$43 | СНГ, 20+ языков | Да | Базовый | Нет | Нет |
| **HookahPlus** | 3 тарифа (цены скрыты) | США | Интеграция Square/Clover/Toast | Нет | Да (сессии) | Нет |
| **Bepoz** | подписка (цена скрыта) | США | Да | Базовый | Нет | Нет |
| **RPOWER** | по запросу | США | Да | Базовый | Нет | Нет |
| **Clover (hookah)** | $14.95–$94.85 | США | Да | Нет | Нет | Нет |
| **Hookah Torus** | **Free / Pro €29 / Enterprise €79** | **DACH → Global** | **ready2order** | **Да, по граммам** | **Да + AI** | **Да** |

**Ключевые выводы:**
1. **Ни один конкурент не работает в Германии** — Kavapp/SkyService ориентированы на СНГ, HookahPlus/Bepoz/RPOWER на США
2. **Ни у кого нет маркетплейса табака** — мы единственные с commerce
3. **Ни у кого нет интеграции с ready2order** — немецкий стандарт POS
4. **Ценовой диапазон** для hookah-софта: $17–$95/мес → наши €29/€79 в рынке

### Наш конкурентный рок (moat)

1. **Единственный hookah SaaS в Германии** — нулевая конкуренция
2. **ready2order интеграция** — уже построена, конкуренты не имеют
3. **Маркетплейс** — софт + commerce = высокий lock-in
4. **Данные** — знаем какие миксы популярны → оптимизация закупок
5. **TSE-compliant** через ready2order → для баров это must-have
