**Hookah Torus** — B2B-Software für Shisha-Bars

- Bestandsverwaltung, Rezepturen, Kassensystem-Integration
- Live: https://hookahtorus.com

---

### FOLIE 2: Das Problem

**4.000–5.000 Shisha-Bars in Deutschland haben keine branchenspezifische Software.**

- Tabakbestand wird per Excel oder Papier geführt
- Kein Kassensystem kann Tabakverbrauch pro Gramm erfassen
- Keine Verbindung zwischen Kasse und Warenwirtschaft
- Betreiber kennen ihre tatsächlichen Kosten pro Sitzung nicht
- Steuerliche Kontrollen: >50% der Bars fallen durch

---

### FOLIE 3: Die Lösung

**Hookah Torus** bietet alles in einer Plattform:

| Funktion | Nutzen |
|---|---|
| Tabak-Inventar (auf Gramm) | Automatischer Bestandsabgleich |
| Mischungsrechner | Kostenoptimierung pro Sitzung |
| POS-Integration (ready2order) | Verkauf → automatische Bestandsreduzierung |
| Tischplan | Echtzeit-Überblick im Gastraum |
| Marktplatz | Tabak direkt nachbestellen |
| Gäste-Kartei | Stammkunden wiedererkennen |

### FOLIE 5: Geschäftsmodell

**Zwei Umsatzströme:**

```
1. SaaS-Abonnements
   Free (€0) → Pro (€29/M) → Enterprise (€79/M)

2. Marktplatz-Provisionen
   8-15% auf Tabakbestellungen über die Plattform
```

**Beispiel: 100 zahlende Bars**
- SaaS: 100 × €35 avg × 12 = €42.000/Jahr
